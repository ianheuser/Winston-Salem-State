# Winston-Salem State Graduate College

Static HTML, CSS, and JavaScript build from the provided Figma file.

## Pages

- `index.html` - Graduate College landing page
- `program.html` - Reusable program page template populated with the MCST mockup content

## Local Preview

Open `index.html` directly in a browser, or run a small local server from this folder:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
